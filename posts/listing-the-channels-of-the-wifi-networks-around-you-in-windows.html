Listing the channels of the WIFI networks around you in Windows

Published on Tuesday, 11 March 2014

  • CommandLine
  • Migrated
  • Wifi
  • Windows

To list the network channels in use by the various wi-fi networks, you can always use this command in an Administrator command prompt.

netsh wlan show all
